The image above is applied to the <header> element as a background, using the following background properties. The default value of "scroll" is set to "fixed" for the background-attachment, causing it to stay in place as the page is scrolled:

			
header {
	background-color: #637782;
	background-image: url('img/2000_1500.png');
	background-repeat: no-repeat;
	background-attachment: fixed; 
	background-size: cover;
}

This can be shorthanded as well, by listing color, image, repeat, position, and attachment in order. Any of these can be skipped but they must be in this order.

The background-size rule is a bit newer and not part of the shorthand; it has to be added separately:

		
header {
	background: #637782 url('img/2000_1500.png') no-repeat center fixed; 
	background-size: cover;
}
