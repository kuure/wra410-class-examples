Form Elements

Examples of most/all of the possible HTML form elements are shown below, grouped with the <fieldset> tag. These are all the ways in which to gather user input within a browser; you must still process the contents of these fields using JavaScript or another language on the server side in order to actually do anything with these values.

All the styling is the (ugly) default from the browser, other than adding a margin: 3rem; to all <fieldset> tags to space them apart from each other as well as a margin-bottom: 1rem; to each <div> tag (which are used to group inputs and their labels inside each fieldset).

Different Types of <input> elements:
Free-Form Text Area
Drop-Down Menus
Checkboxes - allow for multiple selections within a category
Radio Boxes - allow for only a single selection within a category
Button Types - Using both the <button> and the <input> tags
Number and Range Inputs
Date and Time - these are a bit weirder and less supported
