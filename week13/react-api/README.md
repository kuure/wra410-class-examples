# API

This is a simple example of retrieving data from an API in a React
application.

## Installation

To install, type `npm install` at the command line.

The main application code is in `/src/App.jsx`. 

To start the server, type `npn run dev`; you can then visit
<http://localhost:5173> to see your site. Any changes you make to the
source files should automatically reload in the browser. 

To quit, type CTRL-C in the terminal
